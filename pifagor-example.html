Етап 3 · Приклад

Розглянемо приклад

Застосуємо теорему Піфагора до конкретного прямокутного трикутника. Спочатку запишемо катети, а тоді за формулою знайдемо гіпотенузу. Той самий порядок дій підійде для будь-якого прямокутного трикутника.

Теорема Піфагора$c^2 = a^2 + b^2$
Катет (якщо відома гіпотенуза)$a = \sqrt{c^2 - b^2}$

Покроковий розв'язок

Катети 6 і 8 — знайти гіпотенузу
  1. 1Записуємо катети: $a = 6$, $b = 8$
  2. 2За теоремою: $c^2 = 6^2 + 8^2 = 36 + 64 =$ 100
  3. 3Добуваємо корінь: $c = \sqrt{100} =$ 10
Відповідь: гіпотенуза $c = 10$

Подивись, як це рахується крок за кроком

Обчислення гіпотенузи · демонстраціятільки перегляд
  1. 1.$a = 6$, $b = 8$ (катети)
  2. 2.$c^2 = a^2 + b^2$
  3. 3.$c^2 = 6^2 + 8^2$
  4. 4.$c^2 = 36 + 64$
  5. 5.$c^2 = 100$
  6. 6.$c = \sqrt{100} = 10$
Рахує c для трикутника вище. Значення змінити не можна.

Що показує ця демонстрація. Вона проходить ті самі кроки, що й ти на папері: підставляє катети у формулу та рахує гіпотенузу. Так теорема Піфагора видно як чіткий алгоритм — його можна повторити для будь-якого прямокутного трикутника.
