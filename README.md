# Графи знань «Квант» — публічний сайт

Статичний сайт з інтерактивними графами знань з математики (програма НУШ).
Розміщується на GitHub Pages як самодостатні HTML-файли (без збірки, без залежностей).

## Вміст

| Файл | Що це |
|------|-------|
| `index.html` | Титульна сторінка з посиланнями на графи |
| `math.html` | Граф усієї математики 5–11 класи (95 тем, 162 зв'язки) |
| `fractions.html` | Детальний пілот теми «Дроби» (2–6 класи) |

## Як оновити

Файли генеруються з монорепозиторію платформи «Квант»:

```bash
node packages/knowledge-graph/viz/build.ts
```

Скрипт перезаписує `math.html` і `fractions.html` у цій теці. Після цього:

```bash
git add -A && git commit -m "Оновлення графів" && git push
```

GitHub Pages передеплоїть автоматично.
